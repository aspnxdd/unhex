A NodeJS package to convert any RGB color to HEX, HSL color or viceversa.

### Example

```css
main {
   color: #fff;
   background-color: #0070f3;
}
```

After running `npx unhex@latest` with `hexToRgb` direction.

```css
main {
   color: rgb(255, 255, 255);
   background-color: rgb(0, 112, 243);
}
```
